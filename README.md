# Social Hub

🌐 Modern Social Hub & Digital Business Card for Web Developer.

![Profile Preview](assets/profile.png)

## 🇮🇩 Tentang Proyek

Social Hub adalah website profil digital yang dirancang untuk menghubungkan seluruh platform online dalam satu halaman yang sederhana, modern, dan profesional.

Website ini berfungsi sebagai pusat akses menuju:

* Personal Website
* WhatsApp
* GitHub
* Instagram
* Facebook
* Email

Dibangun dengan pendekatan modern untuk memperkuat personal branding dan mempermudah pengunjung terhubung melalui berbagai platform digital.

---

## 🇬🇧 About Project

Social Hub is a modern digital profile website designed to centralize online platforms into a single professional page.

This website provides quick access to:

* Personal Website
* WhatsApp
* GitHub
* Instagram
* Facebook
* Email

Built to strengthen personal branding and provide visitors with a simple way to connect across multiple digital platforms.
